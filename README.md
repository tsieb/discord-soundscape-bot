# Discord Soundscape Bot

## Starter Sound Pack

Run the generator to create a small starter library in `sounds/generated/`:

```bash
npm run generate-sounds
```

The script generates 18 short MP3 files across:

- `sounds/generated/tones/`
- `sounds/generated/noise/`
- `sounds/generated/chimes/`

## Adding More Sounds

You can add your own clips by dropping supported files into `sounds/` (or nested
folders for categories), then running `/sounds list` in Discord.

Useful sources:

- [freesound.org](https://freesound.org/)
- [soundbible.com](http://soundbible.com/)
- [pixabay sound effects](https://pixabay.com/sound-effects/)

Before adding third-party sounds, verify their license terms for your use case.
